STRUCTURE

三層構造（可変型）

LAYER 03
投資層 ファンドSPC / 匿名組合

国内外の投資家からの出資を受け入れる層。SPVへ資金を供給し、リターンを分配。投資家の入替・追加が自由。

日本投資家 海外富裕層 法人出資 REIT化可能
↓ 出資↑ 配当
LAYER 02
資産保有層 開発SPC（合同会社）

不動産・建物・設備を全て保有。倒産隔離により運営リスクから資産を保護。売却・担保化・REIT化の全てに対応。

土地 建物 附属設備 什器備品
↓ 賃貸借↑ 賃料
LAYER 01
運営層 運営会社（あなた）

施設運営・管理・利用者対応に集中。資産を持たないことでBSを軽量化し、倒産リスクを隔離。全国展開に最適。

資産なし 運営管理 入居者収益 BS超軽量
なぜ三層構造が最強か：スキーム変更時にBS組替不要。自社保有 → 投資家売却 → SPV分離 → REIT化 → 海外展開、全てこの構造のまま移行可能。一般的な不動産会社・コンサルがここを理解していないケースが多い。
PATTERNS

5つの展開パターン

PATTERN 01

自社保有型

SPC = 自社100%所有。全資産オンバランス。銀行評価UP、担保化可能。王道パターン。

資産計上オンバランス
BS影響全額計上
メリット融資最大化・担保力
PATTERN 02

投資家売却型

SPV持分を売却。建物ではなく会社を売る。譲渡税軽減、M&A扱いで出口が楽。

資産計上ON → 売却時OFF
税務譲渡税軽減
メリット出口戦略・含み益実現
PATTERN 03

SPV分離型（推奨）

運営のみ自社、資産は投資家SPV。超軽量BSで全国展開が可能に。

資産計上オフバランス
BS影響運営のみ
メリット多拠点展開・リスク分離
PATTERN 04

REIT化ルート

複数SPVを束ねて私募REIT/介護REIT化。出口規模は数百億。

資産計上オフバランス
出口規模数百億〜
目安30棟〜100棟
PATTERN 05

海外投資家型

SPV出資で海外投資家を受入。不動産直接所有より圧倒的に簡単。

対象SG / HK / UAE / US
1室投資¥3,000万 × 100室 = 30億
付加価値オーナービザ・利回り
